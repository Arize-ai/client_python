# MonitorStatus

The monitor's current state from its most recent evaluation. - `TRIGGERED` - The metric breached the threshold. - `CLEARED` - The metric is within healthy bounds. - `NO_DATA` - No data was available to evaluate. 

## Enum

* `TRIGGERED` (value: `'TRIGGERED'`)

* `CLEARED` (value: `'CLEARED'`)

* `NO_DATA` (value: `'NO_DATA'`)
