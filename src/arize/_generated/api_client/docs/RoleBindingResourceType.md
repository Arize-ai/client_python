# RoleBindingResourceType

Resource type for the binding. Only `SPACE` and `PROJECT` are supported for single-binding CRUD. `resource_id` must encode the same resource type. 

## Enum

* `SPACE` (value: `'SPACE'`)

* `PROJECT` (value: `'PROJECT'`)
