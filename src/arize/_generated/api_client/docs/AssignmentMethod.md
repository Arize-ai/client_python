# AssignmentMethod

How records are assigned to annotators. Defaults to \"all\". - `all`: Every annotator is assigned to every record. - `random`: Each record is randomly assigned to one annotator. 

## Enum

* `ALL` (value: `'all'`)

* `RANDOM` (value: `'random'`)
