# DataGranularity

Data granularity level for evaluation. - SPAN - Evaluate at the individual span level. - TRACE - Evaluate at the full trace level. - SESSION - Evaluate at the session level.

* `SPAN` (value: `'SPAN'`)

* `TRACE` (value: `'TRACE'`)
