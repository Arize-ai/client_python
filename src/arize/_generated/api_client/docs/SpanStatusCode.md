# SpanStatusCode

Status code of the span. - OK - The operation completed successfully. - ERROR - The operation failed with an error. - UNSET - No status code was set (default, treated as OK). 

## Enum

* `OK` (value: `'OK'`)

* `ERROR` (value: `'ERROR'`)

* `UNSET` (value: `'UNSET'`)
