# ThresholdCalculation

The statistical calculation used to derive a dynamic threshold. - `STDEV` - standard deviation of the metric over the baseline. - `MAD` - median absolute deviation of the metric over the baseline. 

## Enum

* `STDEV` (value: `'STDEV'`)

* `MAD` (value: `'MAD'`)
