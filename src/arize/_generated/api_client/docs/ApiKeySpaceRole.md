# ApiKeySpaceRole

Space-level role for a service key's bot user. - admin - Full control within the space. - member - Standard space access. - read-only - View-only access. 

## Enum

* `ADMIN` (value: `'admin'`)

* `MEMBER` (value: `'member'`)

* `READ_MINUS_ONLY` (value: `'read-only'`)
