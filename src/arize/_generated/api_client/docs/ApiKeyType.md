# ApiKeyType

Type of the API key. - USER - Personal key that authenticates as the creating user. - SERVICE - Key that authenticates as a service account with explicitly granted access. 

## Enum

* `USER` (value: `'USER'`)

* `SERVICE` (value: `'SERVICE'`)
